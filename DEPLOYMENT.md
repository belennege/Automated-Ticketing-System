# Deployment Guidance

## Installation
- Python 3.x required
- Clone repository, optional virtual environment

## Running the Program
- Open terminal → navigate to folder → `python main.py`

## Maintenance
- Add new stations/zones in `stations.py`
- Update `fare_calculator.py` if fare rules change
- Test with `main.py` after changes

## Notes
- Terminal warnings like `<prefix>` are Python-specific and harmless
